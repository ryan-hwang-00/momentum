## 고려된 블로그  정리

### velog

#### 특징

- 운영: 벨로그
- 개발자들이 많이 사용함
- 코드 블럭의 사용이 편하다고 함

#### 단점

- 사용하는 사람이 별로 없음
- 광고를 달 수 없음.(나중에 추가된다고 함)

### tistory

#### 특징

- 운영: 카카오
- 대중적임
- 범용적임
- 
#### 단점

- 코드 블럭 등이 좀 불편하다고 함

### github pages

#### 특징

- 운영: github
- 높은 자유도

#### 단점

- 난이도가 너무너무너무 높음
